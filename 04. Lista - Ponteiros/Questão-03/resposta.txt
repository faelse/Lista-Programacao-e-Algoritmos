3)  Na  documentação  são  feitas  várias  referências  aos  termos  na  tabela  abaixo.
Os contratantes pediram que você faça uma breve explicação de cada um deles: 


• Um ponteiro 

R : É uma variável que armazena o endereço de memória de outra variável.
    Ex: int *p; significa que p é um ponteiro para um inteiro.


• Um ponteiro para ponteiro

R : É uma variável que armazena o endereço de outro ponteiro.
    Ex : int **pp; -> pp aponta para um ponteiro p.


• O endereço de um ponteiro 

R : Assim como qualquer variável, um ponteiro também fica 
    armazenado em um endereço da memória.
    Podemos obter usando: &p.
    

• O conteúdo de um ponteiro 

R : É o valor que está no endereço para o qual ele aponta.
    Ex: se p = &x, então *p acessa o valor de x.


• O endereço de uma variável 

R : É o local da memória onde a variável foi armazenada.
    Podemos obter usando &variavel.
